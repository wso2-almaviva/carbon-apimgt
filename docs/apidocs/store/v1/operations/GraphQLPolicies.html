GraphQL Policies

Get the complexity related details of an API

GET https://apis.wso2.com/api/am/store/v1/apis/{apiId}/graphql-policies/complexity

This operation can be used to retrieve complexity related details belonging to an API by providing the API id.

OAuth 2.0 Scope

Not required

Examples

Parameters

Type Name Description Schema Default
Path

apiId

required

**API ID** consisting of the **UUID** of the API.

String

Responses

HTTP Code Description Schema

200

OK. Requested complexity details returned.

404

Not Found. Requested API does not contain any complexity details.
